How copy and move files around
You already learned alot. But do you know how to move, copy and rename files? This should come up handy one day.
Go to /tmp directory and look around.
Create a backup of file 'textfile'. Name it 'textfile_bak'.
Edit the file 'textfile' and change the text. You can write anything you want. You can use the edit command and the file will be opened in Editor.
Copy the backup file into 'backup' directory. If the second argument of the cp> or mv command is a folder, the file is moved or copied into it
Move 'binfile' file to backup'.

Copying and moving files

The copy command (cp) is simple. You just use specified the source (file or directory you wish to copy) as the first argument and the destination (the location and new or old name of the file or directory) as the second argument. This is the same for moving.
In Linux/Mac/Linux you can use the same commands as you learned here.
