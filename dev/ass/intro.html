This terminal uses it's own commands but they are very similar to Linux/Unix like commands. There are tons of different tutorials how to use them.

Every operating system has its own terminal. Unix-like operating systems (GNU/Linux, BSDs, MacOSX) use virtual consoles (Gnome uses Terminal, KDE Konsole etc).

If you are using Windows, you can also use a terminal. Go to your START menu and search for 'Command Prompt'. Type 'cmd' and be amazed. Windows 7 and 8 include PowerShell (which is a bit different, but more powerfull than Command Prompt).

If you are interested, you can read more about this topic in a Wikipedia article: Command-line interface.
