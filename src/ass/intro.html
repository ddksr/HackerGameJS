Type something in the terminal. The terminal is the big black screen with an blinking cursor located on your left. When you are finished typing press ENTER on your keyboard.
Make the terminal respond. Use the 'echo' command and try making the terminal respond 'Hello world!'. Try not to use hints because you will receive penalty points. Type 'help echo' in the terminal. The 'help' command can help you understand any of the terminal commands you will ever need to use. Type 'echo Hello World' in the terminal.
Send a message to your sensei. The command is called 'sensei' and is simple to use. Use the 'help' command to learn how to use it and send a test message. The content of the message isn't important. The 'help' command will always help you when you are stuck with a command or a program you never used. Type 'help sensei' First type 'help sensei' and then 'sensei my message'

The help command is not just used for explaining one command. You can use it to list all the commands that are available on this computer. Try it just by typing help.
Don't ever forget that this is your computer which contains programs, files and directories (folders). Type ls to list all the files and directories that you can currently see.

This terminal uses it's own commands but they are very similar to Linux/Unix like commands. There are tons of different tutorials how to use them.

Every operating system has its own terminal. Unix-like operating systems (GNU/Linux, BSDs, MacOSX) use virtual consoles (Gnome uses Terminal, KDE Konsole etc).

If you are using Windows, you can also use a terminal. Go to your START menu and search for 'Command Prompt'. Type 'cmd' and be amazed. Windows 7 and 8 include PowerShell (which is a bit different, but more powerful than Command Prompt).

If you are interested, you can read more about this topic in a Wikipedia article: Command-line interface.
